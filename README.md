# ☕ Algorithm Study

알고리즘 문제 풀이 및 학습 기록을 위한 리포지토리입니다.

기존에는 **Python**을 사용했으나, 현재는 **Java**를 사용하여 문제를 해결하고 있습니다.

## 📂 Directory Structure

각 플랫폼별로 폴더를 구분하여 관리합니다.

```text
Algorithm-Study/
├── 📂 Baekjoon         # 백준 (유형별 분류)
├── 📂 Programmers      # 프로그래머스 (기출/유형별 분류)
└── 📂 SWEA             # SW Expert Academy
```

## 🛠️ Environment & Languages

- **Main Language**: Java 11+
- **Legacy**: Python
- **IDE**: IntelliJ IDEA, VS Code

## 📏 File Naming Rules

파일 생성 시 다음과 같은 규칙을 따릅니다.

|     플랫폼      | 파일명 규칙                | 예시                        |
| :-------------: | :------------------------- | :-------------------------- |
|  **Baekjoon**   | `BOJ_{문제번호}.{확장자}`  | `BOJ_14940.java`            |
|    **SWEA**     | `SWEA_{문제번호}.{확장자}` | `SWEA_1208.java`            |
| **Programmers** | `{문제이름}.{확장자}`      | `두 큐 합 같게 만들기.java` |

## 📝 Commit Message

커밋 메시지는 다음 형식을 따릅니다.

**Format**: `[Platform] {문제이름} _{난이도}`

**Example:**

- `[Prog] 두 큐 합 같게 만들기 _Lv.2`
- `[BOJ] 쉬운 최단거리 _Silver1`
